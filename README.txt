🎱 Bankline Billiards – Digital Pool Prototype
Bankline Billiards is a lightweight, single-player billiards-inspired game prototype built in Python using the Pygame library.
Originally designed as an experimental project, it explores real-time physics, user interaction, file persistence, and AI gameplay logic.

🔧 Features
🎯 2D billiards simulation with realistic physics

🚧 Collision handling with bumpers, pockets, and posts

🧠 Practice mode and AI opponent mode

📝 Trick shot save/load system (stores shots to file)

🎮 Score tracking and foul detection

💾 Player name entry and menu navigation

⚙️ Clean object motion with velocity damping

🪶 Lightweight standalone MVP (no external dependencies)

🛠 Tech Stack
Python 3.x

Pygame
